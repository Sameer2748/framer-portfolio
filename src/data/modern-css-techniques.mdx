# Modern CSS Techniques and Best Practices

## The Evolution of CSS

Modern CSS has evolved significantly, offering powerful features that make web styling more flexible and maintainable than ever before. Let's explore the most important modern CSS techniques.

## CSS Grid Layout

### Basic Grid
```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}
```

### Grid Areas
```css
.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}
```

## Flexbox

### Modern Card Layout
```css
.card-container {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
  justify-content: center;
}

.card {
  flex: 1 1 300px;
  max-width: 400px;
}
```

## CSS Custom Properties

### Theme Implementation
```css
:root {
  --primary-color: #007bff;
  --secondary-color: #6c757d;
  --spacing-unit: 8px;
}

.button {
  background-color: var(--primary-color);
  padding: calc(var(--spacing-unit) * 2);
}
```

## Modern Selectors

### Advanced Selectors
```css
/* Direct children */
.container > * {
  margin-bottom: 1rem;
}

/* Siblings */
.card + .card {
  margin-left: 1rem;
}

/* State-based */
.input:has(:focus) {
  border-color: var(--primary-color);
}
```

## CSS Animations

### Modern Animations
```css
.fade-in {
  animation: fadeIn 0.3s ease-in;
}

@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

## Container Queries

### Responsive Components
```css
.card {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card-content {
    display: grid;
    grid-template-columns: 2fr 1fr;
  }
}
```

## CSS-in-JS Integration

### Styled Components Example
```typescript
const Button = styled.button`
  background: ${props => props.primary ? 'var(--primary-color)' : 'white'};
  color: ${props => props.primary ? 'white' : 'var(--primary-color)'};
  padding: 0.5em 1em;
  border: 2px solid var(--primary-color);
  border-radius: 4px;
`;
```

## Performance Optimization

1. **Will-change Property**
```css
.animated-element {
  will-change: transform;
  transition: transform 0.3s ease;
}
```

2. **Content-visibility**
```css
.long-content {
  content-visibility: auto;
  contain-intrinsic-size: 0 500px;
}
```

## Best Practices

1. **Mobile-First Approach**
```css
.element {
  /* Base styles for mobile */
  width: 100%;
  
  /* Tablet and up */
  @media (min-width: 768px) {
    width: 50%;
  }
  
  /* Desktop */
  @media (min-width: 1024px) {
    width: 33.333%;
  }
}
```

2. **Logical Properties**
```css
.container {
  margin-block: 1rem;
  padding-inline: 2rem;
}
```

## Conclusion

Modern CSS provides powerful tools for creating responsive, maintainable, and performant web applications. Understanding and utilizing these features is crucial for modern web development.
